# Installation

The entire stack — backend, database, message broker, Wazuh SIEM, dashboard, ML training, and telemetry simulator — is bootstrapped by a single script.

## 1. Clone the monorepo

<Snippet title="Clone the repository" value="git clone https://github.com/PeakChainNetwork/peaksoft-security-platform.git && cd peaksoft-security-platform" type="shell">
{`$ git clone https://github.com/PeakChainNetwork/peaksoft-security-platform.git\n$ cd peaksoft-security-platform`}
</Snippet>

## 2. Run the setup script

<Snippet title="One-click setup" value="bash scripts/setup.sh" type="shell">
{`$ bash scripts/setup.sh`}
</Snippet>

<Callout title="What the script does" variant="info">
This single command handles the entire bootstrap sequence automatically. Sit back — it takes a few minutes on first run.
</Callout>
