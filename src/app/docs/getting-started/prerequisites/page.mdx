# Prerequisites

The entire platform runs as a Docker stack orchestrated by a single setup script. You only need two things installed on your machine.

## Required

<Callout title="That's it" variant="info">
You do **not** need to install Node.js, Python, pnpm, or any backend dependencies locally. Everything runs inside Docker containers managed by <code>scripts/setup.sh</code>.
</Callout>
