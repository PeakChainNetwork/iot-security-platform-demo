## Regenerating the API snapshot

After backend route or schema changes, run from the repo root (with the backend virtualenv active):

<Snippet title="Export OpenAPI snapshot" value="pnpm docs:openapi" type="shell">
{`$ pnpm docs:openapi`}
</Snippet>

This overwrites `public/openapi.json` using `iot-security-backend/scripts/export_openapi.py`.
